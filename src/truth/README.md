# truth

- Per-relationship truthiness (seeded from end-of-transcript true/false judgments).
- Propagation: if X says Y and X is judged unlikely, lower Y's likelihood.
- Contradiction detection: two truths or two falsehoods that can't coexist.
- Loop detection: logic cycles in the graph.
- Novel-link surfacing: implicit relationships nobody stated outright
  (e.g. X was at both events).
